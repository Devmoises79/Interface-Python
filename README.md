# 🖥 Interface Python — Sistema de Login (CustomTkinter)

📖 Sobre o Projeto

Este projeto implementa uma interface gráfica simples de login utilizando a biblioteca CustomTkinter, com foco em criação de aplicações desktop modernas em Python.

O sistema valida credenciais inseridas pelo usuário e retorna mensagens visuais de sucesso ou erro diretamente na interface.

# 🎯 Objetivos

- Praticar criação de interfaces gráficas modernas
- Utilizar a biblioteca CustomTkinter
- Implementar lógica de autenticação simples
- Trabalhar com eventos e callbacks em GUI
- Reforçar estruturação de aplicações desktop

# ⚙️ Tecnologias Utilizadas

- Python 3
- CustomTkinter

# 🧩 Funcionalidades

- 👤 Campo de entrada para usuário
- 🔒 Campo de entrada para senha
- 🔘 Botão de login com evento
- ✔ Validação de credenciais
- 🎨 Feedback visual (sucesso ou erro)
- 🌙 Tema escuro da interface
- 🔐 Lógica de Funcionamento

O sistema valida os dados diretamente no código:

```bash
if usuario == 'Moisés' and senha == 'Moises123':
```

- Se as credenciais forem corretas:  → login autorizado
- Caso contrário: → acesso negado

# 📌 Exemplo de Uso

```text
Usuário: Moisés
Senha: Moises123

Resultado:

Login feito com sucesso! (verde)
```

# 🧠 Aprendizados

Este projeto reforça conceitos importantes:

- Criação de interfaces gráficas com Python
- Uso de componentes (labels, inputs, botões)
- Manipulação de eventos (callbacks)
- Lógica condicional aplicada a UI
- Estruturação de aplicações desktop

```bash
pip install customtkinter

```
